# Coffee Delivery

Aplicação desenvolvida no módulo "Ignite" proposto pela RocketSeat.

🛠️ Skils: TypeScript, React, Styled Components HTML, CSS


## 📋 Clone o repositório
$ git clone https://github.com/HiLucred/Coffee-Delivery-REACT


## 🔧 Instale as dependências
$ npm install

 
## ⚙️ Rode o projeto
$ npm run dev
